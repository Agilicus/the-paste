The Paste
==========

Paste image data into the Editor and upload it to the WordPress Media Library.

 - Developed in WP 3.8 – WP 4.8
 - Should work with WordPress 3.5+
 - German and Dutch localization. 

Known Browser Support
---------------------

 - Chrome 32+
 - Firefox 26+
 - Safari 10+
 - IE11
 - Edge
